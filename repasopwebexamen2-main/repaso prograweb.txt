Descarga Node.js en -> https://nodejs.org/en

 

Instalacion de express (https://expressjs.com/es/starter/installing.html)

 

$ mkdir myapp

$ cd myapp

$ npm init

$ npm install express

# Para instalar Express temporalmente y no añadirlo a la lista de dependencias, omita la opción --save abajito:

$ npm install express --no-save

 

 

Instalacion de Vite (https://vite.dev/guide/)

 

$ npm create vite@latest

$ cd vite-project

$ npm install

$ npm run dev

 

Instalacion de dependencias del backend:

 

$ npm install express dotenv body-parser cors            # runtime

$ npm install -D typescript ts-node-dev                  # build / reload

$ npm install -D @types/node @types/express

$ npm install -D @types/body-parser @types/cors          # .d.ts files




=================================================================================================================================================================================================================



Instalacion de dependencias del front:

 

npm install react react-dom react-router-dom bootstrap axios

npm install -D @types/react @types/react-dom @types/react-router-dom
